# Intraday-Iron-Condor-Options-Data-Analysis
Analyzing hourly stock data to locate potential trading edges using an iron condor options strategy  

# Data Set Information
Data is sourced via Kaggle by user geowtt: https://www.kaggle.com/datasets/geowtt/hourly-stock-prices-technical-indicators-2023
Dataset follows a Attribution 4.0 International License, as described here: https://creativecommons.org/licenses/by/4.0

As described on Kaggle by geowtt: 

"This dataset contains hourly price data (OHLCV) and common technical indicators
for 8 major U.S. tickers across technology, finance, energy, and automotive sectors.

Indicators include SMA (10/50), EMA (20), RSI (14), MACD (+signal, +histogram), and volatility (20).
The target column target_up_next indicates whether the next hourly close
was at least 0.05% higher than the current one."
